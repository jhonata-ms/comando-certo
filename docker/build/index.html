No parâmetro -f informe o caminho do Dockerfile.

No parâmetro -t informe o nome que terá sua imagem quando for "buildada".

No final do comando informe a pasta com código fonte do seu projeto.

Exemplo:

docker build -f Dockerfile -t jhonata-ms/minha-aplicacao:latest /opt/pasta/do/projeto

Caso já estiver na pasta do projeto, que contém o Dockerfile e os códigos fontes, então pode resumir o comando, passando somente um ponto no lugar do path completo, veja:

docker build -t jhonatams/minha-aplicacao:latest .
